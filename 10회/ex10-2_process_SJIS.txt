基本課題 2. 計算過程回答ファイル

以下の入力例を参考にして、n = 4 における
プログラムの実行結果と検算結果、
手計算で検算した計算過程を回答欄に入力して
アップロードしてください。

計算過程の入力例) n = 3 の場合

f( 3 )
= 3 * g( 2 )
= 3 * ( 2 + f( 1 ) )
= 3 * ( 2 + 1 )
= 9

g( 3 )
= 3 + f( 2 )
= 3 + ( 2 * g( 1 ) )
= 3 + ( 2 * ( 1 + f( 0 ) ) )
= 3 + ( 2 * ( 1 + 1 ) )
= 7
